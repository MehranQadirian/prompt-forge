# PillTabBar Interaction Refactor

## Goal
Refine pressed-state behavior and interaction quality. No layout/visual/navigation changes.

## File to Modify
`src/components/PillTabBar.tsx`

## Changes

### 1. Add `hitSlop={8}` to tab items
Consistent with every other interactive component in the app.

### 2. Refine pressed opacity (0.7 → 0.85)
Current 0.7 is too aggressive for a premium feel. 0.85 is subtle and native-like.

### 3. Add background tint on press
Add `c.onBackground + '08'` background tint on press-in, clear on press-out. This makes the pressed state feel physically attached to the pill shape — the user sees the exact pill area darken.

### 4. Use `TOUCH_TARGET` instead of hardcoded `48`
`styles.tabItem.height: 48` → `TOUCH_TARGET`

### 5. Make `PillTabItem` use `useTheme()` internally
Remove `colors` prop, use `useTheme()` like every other component. Cleaner API, consistent pattern.

### 6. Add `accessibilityHint` to tab items
`accessibilityHint={`Switch to ${tab.label} tab`}`

## Detailed Implementation

### PillTabItem changes

**Before:**
```tsx
function PillTabItem({ tab, isActive, onPress, colors, index, activeIndex, tabWidth }) {
  // ...
  return (
    <AnimatedPressable
      android_ripple={{ color: colors.onBackground + '14' }}
      style={[styles.tabItem, { width: tabWidth, borderRadius: RADIUS.full }, pressStyle]}
    >
```

**After:**
```tsx
function PillTabItem({ tab, isActive, onPress, index, activeIndex, tabWidth }) {
  const { theme } = useTheme();
  const c = theme.color;

  const pressedOpacity = useSharedValue(1);
  const pressedBg = useSharedValue(0); // 0 = transparent, 1 = tinted

  const handlePressIn = useCallback(() => {
    pressedOpacity.value = withTiming(0.85, { duration: 120 });
    pressedBg.value = withTiming(1, { duration: 120 });
  }, []);

  const handlePressOut = useCallback(() => {
    pressedOpacity.value = withTiming(1, { duration: 120 });
    pressedBg.value = withTiming(0, { duration: 120 });
  }, []);

  const pressStyle = useAnimatedStyle(() => ({
    opacity: pressedOpacity.value,
    backgroundColor: pressedBg.value === 1 ? c.onBackground + '08' : 'transparent',
  }));

  return (
    <AnimatedPressable
      onPress={...}
      onPressIn={handlePressIn}
      onPressOut={handlePressOut}
      hitSlop={8}
      accessibilityRole="button"
      accessibilityLabel={...}
      accessibilityHint={`Switch to ${tab.label} tab`}
      accessibilityState={{ selected: isActive }}
      android_ripple={{ color: c.onBackground + '14' }}
      style={[styles.tabItem, { width: tabWidth, borderRadius: RADIUS.full }, pressStyle]}
    >
```

### PillTabBar — remove `colors` prop from PillTabItem calls
```tsx
// Before:
<PillTabItem tab={tab} isActive={...} onPress={...} colors={c} index={...} activeIndex={...} tabWidth={...} />

// After:
<PillTabItem tab={tab} isActive={...} onPress={...} index={...} activeIndex={...} tabWidth={...} />
```

### styles.tabItem — use TOUCH_TARGET
```tsx
tabItem: {
  alignItems: 'center',
  justifyContent: 'center',
  height: TOUCH_TARGET,  // was 48
  zIndex: 1,
},
```
